Add your photos here (JPG, JPEG, PNG, or WebP).

Use these base names (any of those extensions works — the site tries each in order):

  hero       — Hero section, right column (portrait / vertical looks best)
  product-1  — “Meet the headliner” carousel — image 1
  product-2  — carousel — image 2
  product-3  — carousel — image 3
  about      — About section card (optional — if missing, the CC monogram shows instead)

Examples: hero.jpg, hero.png, product-1.webp

After adding files, save and refresh the dev server. For GitHub Pages, commit these files and deploy.
